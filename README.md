# 🐍 PythonLearn

A personal learning repo documenting my Python journey. From fundamentals to real-world projects.

Everything here is a work in progress. The goal isn't perfection, it's consistency.

---

## What's inside

| Folder | Description |
|--------|-------------|
| `python-101` | Core concepts — variables, loops, functions, OOP |
| `python-mini-projects` | Small projects built to apply what I've learned |
| `api-tests` | Experimenting with REST APIs and HTTP requests |
| `challenges` | Coding exercises and problem solving |
| `notes` | Cheatsheets and snippets I keep coming back to |
