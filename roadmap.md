# 🐍 Python Learning Roadmap

A structured path for learning Python — from the basics through to real-world projects.

---

## Stage 1 — The Basics
> Get comfortable with the language fundamentals

- [ ] Variables & data types (`str`, `int`, `float`, `bool`)
- [ ] User input & output (`input()`, `print()`)
- [ ] String manipulation (slicing, methods, f-strings)
- [ ] Arithmetic & comparison operators
- [ ] Comments & code structure

---

## Stage 2 — Control Flow
> Make decisions and repeat actions in code

- [ ] `if`, `elif`, `else` statements
- [ ] Comparison operators (`==`, `!=`, `>`, `<`, `>=`, `<=`)
- [ ] Logical operators (`and`, `or`, `not`)
- [ ] `while` loops
- [ ] `for` loops
- [ ] `break`, `continue`, `pass`
- [ ] `range()`

---

## Stage 3 — Data Structures
> Store and organise data effectively

- [ ] Lists — indexing, slicing, methods, comprehensions
- [ ] Tuples — immutable sequences
- [ ] Dictionaries — key/value pairs, methods
- [ ] Sets — unique values, set operations
- [ ] Nested data structures
- [ ] When to use each structure

---

## Stage 4 — Functions
> Write reusable, organised code

- [ ] Defining and calling functions
- [ ] Parameters & arguments
- [ ] Default parameters
- [ ] Return values
- [ ] `*args` and `**kwargs`
- [ ] Scope — local vs global
- [ ] Lambda functions
- [ ] Recursion basics

---

## Stage 5 — Modules & Packages
> Use Python's built-in tools and third-party libraries

- [ ] Importing modules (`import`, `from ... import`)
- [ ] Standard library — `math`, `random`, `datetime`, `os`, `sys`
- [ ] Installing packages with `pip`
- [ ] Virtual environments (`venv`)
- [ ] Managing dependencies with `requirements.txt`
- [ ] Popular third-party packages — `requests`, `beautifulsoup4`

---

## Stage 6 — File Handling
> Read from and write to files

- [ ] Opening and closing files
- [ ] Reading files (`read()`, `readline()`, `readlines()`)
- [ ] Writing and appending to files
- [ ] Working with `with` statement
- [ ] Working with CSV files
- [ ] Working with JSON files
- [ ] File paths with `os.path` and `pathlib`

---

## Stage 7 — Error Handling
> Write code that handles problems gracefully

- [ ] Understanding exceptions
- [ ] `try`, `except`, `else`, `finally`
- [ ] Catching specific exceptions
- [ ] Raising exceptions with `raise`
- [ ] Custom exceptions
- [ ] Debugging techniques

---

## Stage 8 — Object Oriented Programming (OOP)
> Model real-world problems with classes and objects

- [ ] Classes and objects
- [ ] `__init__` method
- [ ] Instance variables & methods
- [ ] `self`
- [ ] Inheritance
- [ ] Method overriding
- [ ] Encapsulation
- [ ] `__str__` and `__repr__`
- [ ] Class vs instance variables

---

## Stage 9 — Working with APIs
> Connect your code to the outside world

- [ ] What is an API / REST API
- [ ] HTTP methods — `GET`, `POST`, `PUT`, `DELETE`
- [ ] Using the `requests` library
- [ ] Handling JSON responses
- [ ] Query parameters & headers
- [ ] API authentication (API keys, Bearer tokens)
- [ ] Error handling with status codes
- [ ] Building a simple API client

---

## Stage 10 — Web Scraping
> Extract data from websites

- [ ] How web scraping works
- [ ] HTML basics — tags, attributes, structure
- [ ] `requests` + `BeautifulSoup`
- [ ] Navigating the DOM
- [ ] Extracting text, links, and attributes
- [ ] Scraping multiple pages
- [ ] Storing scraped data (CSV, JSON)
- [ ] Respecting `robots.txt`

---

## Stage 11 — Automation & Scripting
> Make repetitive tasks run themselves

- [ ] Automating file management
- [ ] Working with directories (`os`, `shutil`, `pathlib`)
- [ ] Scheduling scripts (cron, `schedule` library)
- [ ] Sending emails with `smtplib`
- [ ] Automating the browser with `selenium`
- [ ] Working with spreadsheets (`openpyxl`, `pandas`)
- [ ] Command line arguments with `argparse`

---

## Stage 12 — Data & Databases
> Store, query, and analyse data

- [ ] SQL basics — `SELECT`, `INSERT`, `UPDATE`, `DELETE`
- [ ] SQLite with Python (`sqlite3`)
- [ ] Introduction to `pandas`
- [ ] Reading and writing CSV / Excel with `pandas`
- [ ] Data cleaning basics
- [ ] Introduction to `matplotlib` for charts

---

## Stage 13 — Testing
> Write code you can trust

- [ ] Why testing matters
- [ ] Writing basic tests with `unittest`
- [ ] Introduction to `pytest`
- [ ] Test structure — arrange, act, assert
- [ ] Mocking with `unittest.mock`
- [ ] Test coverage

---

## Stage 14 — Best Practices
> Write clean, professional Python

- [ ] PEP 8 — Python style guide
- [ ] Meaningful naming conventions
- [ ] Docstrings and comments
- [ ] Code formatting with `black`
- [ ] Linting with `pylint` or `flake8`
- [ ] Type hints
- [ ] Writing readable, maintainable code
- [ ] Git & version control workflow

---
